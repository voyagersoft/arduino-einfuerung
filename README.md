# Arduino Einführung

Kurze Übersicht über die vorhandenen Beispiele:

- `01_blink/01_blink.ino`  
	Lässt die interne LED regelmäßig blinken.

- `02_touch/02_touch.ino`  
	Liest einen Touch-Sensor und schaltet damit die interne LED.

- `03_ampel/03_ampel.ino`  
	Steuert eine einfache Ampel mit drei LEDs; jeder Touch schaltet zum nächsten Ampelzustand.

- `04_rfid/04_rfid.ino`  
	Liest RFID-Karten mit einem PN532 ein, gibt die erkannte UID aus und setzt abhängig davon den LED-Zustand.

- `06_abstand/06_abstand.ino`  
	Liest einen HC-SR04 Abstandssensor und zeigt den gemessenen Abstand auf 8 WS2812-LEDs als Farbbalken.
